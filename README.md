# SWIM (Shear Wave Interference Modulation - 剪切波干涉调制)

Title: `Spatiotemporal Interference of Cutaneous Shear Waves Dictates Tactile Perception in Ultrasonic Haptics`

## 研究目标

研究目标重点并不是单纯简单的提出一种新的调制方式，而是通过实验、仿真和理论分析，来深入的研究在不同调制模式下，效果的差异的根本原因/机理（心理学原因，生理物理学原因等等）。偏底层基础研究（如剪切波传播、马赫锥效应、机械感受器激活），并给出一套新的从表面波传播角度分析和看待的超声触觉形成相关的理论（虽然相关研究已经表明了皮肤表明剪切波传播才是产生触觉更强的主要因素，并且也有研究研究了皮肤表面的表面波的马赫锥会导致触觉扩散模糊，但是仍然缺少一个统一的来指导超声触觉设计的理论）。以及给出从实验结果和形成的理论上分析，想要实现强度更高、精度更高的超声触觉反馈，需要满足的条件。

## 背景信息

### 问题分析

传统的LM调制，虽然通过横向移动超声相控阵焦点，实现触发更多的皮肤受体，结合对焦点移动速度的控制，使得焦点移动速度匹配皮肤表面剪切波波速，从而实现对波前的压缩。通过这两个效应达到了更强的触觉反馈强度。但是这种方法存在触觉反馈精度的问题，焦点超音速（皮肤表面波音速）移动时，会产生马赫锥效应，在皮肤表面扩散开来，导致触觉生成精度差。

### 调制方式设计

我现在有一种离散的超声触觉焦点调制方法，名称为 Discrete Lateral Modulation（DLM）。每个时刻焦点都不进行移动，而是在多个不同位置之间进行瞬间切换。

具体细节为：超声相控阵的聚焦点周期性的在一个圆环上均匀分布的多个离散点之间进行切换，每个周期（例如30Hz的一个周期）内超声聚焦点按顺序在每个聚焦点之间转移，例如我上传的两个图片示意了两个焦点切换和三个焦点切换。当采样的离散点个数趋于无穷个时，这种焦点调制方法等效于圆形轨迹的横向调制（LM_C）。

我还根据LM的论文设计了一种 **LM_L** 的变体版本 **ULM_L** (Unidirectional Lateral Modulation (Linear))，聚焦点不是在两个目标点之间来回移动，而是单向的从A点移动到B点，再瞬移到A点继续直线移动到B点。这种调制方式保证了轨迹上每个点受到的刺激频率和轨迹的循环频率保持一致（都是200Hz），不会出现双向的 **LM_L** 由于每个周期经过轨迹中间每个点两次从而产生的2倍频到400Hz的现象。从而保证了最大程度刺激皮肤受体敏感的频率区间。（可能非原创，其他人应该已经做过了，但是可以用来进行对比试验，作为探针来分析数据和理论）。

### 理论分析

我自己对 DLM、ULM_L、LM_L 和 LM_C 调制的初步分析如下：

- 两点DLM（DLM_2）：当我切换聚焦点的时机足够恰好时，在第一个聚焦点A戳下去，等产生的皮肤表面传播的剪切波刚好传播到B处时，再切换焦点到B点。这样会在B点形成一个很大的压力或者剪切梯度，从而更高效的激活皮肤触觉受体，产生明显的触觉感知。

- 三点DLM（DLM_3）：当离散的聚焦点个数增加到3个及以上时，在DLM_2的基础上，每个焦点产生的表面波还会在圆心处产生一个旋转变化的剪切应变/应力，这个旋转剪切量旋转的频率正好是调制整体调制频率（200Hz），从而同样可以激活皮肤中对剪切敏感的触觉受体）。

- ULM_L：ULM_L 调制方式的每个周期内，焦点会单向的直线从A点移动到B点移动，焦点移动速度匹配皮肤表面剪切波波速，从而实现对波前的持续压缩。

- LM_L：持续移动的 **LM_L** 调制方式的焦点会在焦点移动方向前侧产生一个高压区，在焦点移动方向后侧产生一个低压区。当焦点移动到直线线段的两侧两头时，速度方向突然180° 转变，则会让新产生的高压区和之前尾迹的低压区碰撞，产生一个极大的压力或者剪切的梯度，从而同样产生一种很强大的触觉感知。

- LM_C：平滑沿着圆形轨迹移动的焦点在每一时刻都会在皮肤上激发沿着皮肤方向传播的表面剪切波，和DLM类似，这些剪切波也会在圆心处产生旋转剪切场。


### 超声触觉设备 UMH Device 信息

我们研究开发了一种高能量密度，小尺寸的集成式超声触觉设备 UMH Device（Ultrasound Mid-Air Haptic Device），本研究中所进行的超声触觉实验均是在这台设备的基础上进行的。以下是 UMH 设备的详细结构化描述。

#### 1. 设备物理与形态规格

- **整体外形**：正六边形设计，对边距 90mm。

- **三维尺寸**：最大外形尺寸为 102.2 × 90 × 32.2 mm（包含换能器、PCB、铜柱及接口突出部分）。
- **阵元排布**：
  - 采用六边形最密堆积（Hexagonal Close Packing）。
  - 边长为 5 个阵元，理论满排布为 61 个阵元。
  - 实际排布移除最中心位置（未来预留放置 CMOS 图像传感器），实际搭载 **60个** 超声换能器。

#### 2. 核心硬件与主控参数

- **微控制器 (MCU)**：STM32H750VBT6。
  - 运行频率机制：系统使用 DWT 进行周期计数和微秒级延时，通过 SysTick 和内建定时器（TIM1）同步 DMA 触发。

- **供电系统**：
  - 主输入电压：12V DC。
  - 内部 ADC 监测电路（通过 ADC1 和 ADC3）：实时监测 `3.3V`、`5.0V` 电压轨以及 `VDDA` 和 MCU 内部温度。

#### 3. 超声换能器 (Transducer) 规格

- **型号**：GU1007C-40TR（铝壳开放式空气超声换能器，收发一体）DONGUANINGHAI ELECTRONICS CO.,LTD.

- **物理尺寸**：直径 10mm，高度 7mm。
- **声学参数**：
  - 中心工作频率：40 kHz。
  - 波长 ($\lambda$)：在室温（声速 $343.2\text{ m/s}$）下约为 8.58 mm。
  - 阵元间距 (Transducer Spacing)：10 mm。
  - 指向性 (Directivity)：约 80°。

#### 4. 驱动电路与波形生成 (Bipolar Drive)

- **驱动芯片**：IX4428MTR（双通道高速低边 MOSFET 驱动器）。IXYS Integrated Circuits Division

- **驱动拓扑**：
  - 输入单路信号 (`SIG`)，通过电阻与 ESD 保护二极管进入驱动 IC 的 `INA`，同时通过外围反相网络（或芯片内部逻辑）使 `INB` 获得反相信号。
  - 输出端 `OUTA` 和 `OUTB` 接换能器两端，形成**双极性驱动（Bipolar Drive）**或全桥推挽效果。
  - **驱动电压**：驱动 IC 由 12V (`VDD`) 供电。利用推挽输出，换能器两端峰峰值电压 (VPP) 翻倍，达到 **24 VPP**。

#### 5. 波形输出与 DMA 机制

- **相控阵控制方式**：通过 STM32 的 Timer + DMA（Mem-to-Mem 模式映射到 GPIO ODR 寄存器）实现高精度多通道同步输出。

- **DMA 通道分布**：使用 5 个 DMA 数据流（`DMA1_Stream0~2`, `DMA2_Stream0~1`），分别控制 5 个 GPIO 端口 (`GPIOA` 到 `GPIOE`)。
- **采样率与分辨率**：
  - DMA 采样频率设定为 4 MHz (`DMA_SAMPLING_FREQ`)。
  - 对于 40kHz 的载波，每个超声波周期在缓冲区中被划分为 100 个时间片（Buffer Size = 100）。
  - 相位延迟通过换算为缓冲区的偏移位（`shift_buffer_bits`）来实现，具备纳秒级的相位控制精度。

#### 6. 固件算法与声场控制 (Stimulation)

设备通过计算换能器三维坐标到目标焦点的空间距离差异，实现波束成形（Beamforming）。支持多种声场控制模式（定义在 `stimulation.c` 中）：

- **Focus Point (单点聚焦)**：将声能汇聚于三维空间的一点。
- **Discrete / Linear / Circular 轨迹**：焦点在空间中沿离散点、直线或圆形轨迹进行周期性扫描/移动。
- **TwinTrap (双声阱)**：生成用于声悬浮的 TwinTrap 拓扑（通过几何分割，一半换能器施加额外的 $\pi$ 相位差）。
- **相位刷新率**：控制算法以设定的刺激频率（如 200 Hz，即周期为 5ms）在缓冲区内动态更新各个阵元的相位状态。

#### 7. 通讯协议与控制接口

- **物理接口**：USB Type-C。

- **通讯模式**：USB CDC（虚拟串口）。
- **通信帧结构**：
  - 定制的二进制帧协议，最大帧长 260 字节。
  - **格式**：`[Header1 0xAA] [Header2 0x55] [CMD Type] [Data Length] [Data...] [Checksum] [Tail1 0x0D] [Tail2 0x0A]`。
- **支持的控制命令 (CMD)**：
  - `0x01`：启用/禁用刺激输出。
  - `0x02`：Ping 联机测试。
  - `0x03`：获取设备配置（序列号、版本、阵元数、间距等）。
  - `0x04`：获取设备状态（电压、温度、循环频率、当前模式等）。
  - `0x05`：下发并设置声场控制参数（类型、位置、法向量、半径、频率等）。
  - `0x06`：直接设置底层所有阵元的绝对相位。
  - `0x07`：切换内置的演示模式 (Demo Mode)。
  - `0x08`：获取换能器空间三维坐标信息。

#### 8. 校准与调试功能

- **固件校准 (Calibration)**：
  - 设备内置了 60 个阵元的初始时间延迟校准数组（单位：微秒），用于补偿硬件走线、换能器个体差异等造成的本底相位误差。
  - 可通过按键 (`KEY0`) 或命令进入校准模式，在校准模式下所有通道相位清零。

- **板载指示灯 (LEDs)**：
  - 通过 `GPIOA` 的引脚控制。
  - 包含 1Hz 心跳灯 (`LED0`)、校准模式指示灯 (`LED1`) 以及 Demo 模式状态指示灯 (`LED2`)。



### 实验1

#### 实验1设计

由于每种调制刺激模式产生的触觉感知绝对值较弱，导致相对的对比差异也较弱。为了保证能够获取到足够的正确的对比差异，实验设计采用对比式实验范式。

1. 实验阵列与组合
共有5个自变量（$DLM_2$, $DLM_3$, $ULM_L$, $LM_L$, $LM_C$）。
将它们进行两两组合，总共产生 $C(5,2) = 10$ 个比较对（例如：$DLM_2$ vs $LM_C$）。
考虑到呈现顺序的遗留效应（先出现的刺激会被记忆衰减影响），需要引入正反序。每个Block包含20个试次（Trials）（即10个比较对 $\times$ 2次呈现顺序）。

2. 硬件与物理环境控制
听觉遮蔽：被试必须佩戴播放固定的白噪声的降噪耳机，阻断相控阵开关时产生的轻微声波或高频噪音泄漏导致对被试触觉感知判断失误。
固定与耦合：被试的手掌必须放置在一个固定高度的泡沫垫上，控制手掌到超声换能器间的距离和位置固定。

##### 实验流程（按被试进行）

使用 G*Power 计算实验1需要的样本大小，设置（Effect size g=0.2, alpha err prob=0.5, Power=0.8，Constant proportion = 0.5），计算出的Total sample size = 49，由于每个被试需要进行正反两次呈现顺序，即每个被试会对同一个比较对贡献 2 个 Trials。因此，计算出所需被试数为 $49 / 2 = 25$ 人。

最终招募 30 名 被试进行实验1。性别比例为 1:1（male:female = 13:13）。

实验分为两个独立的Block（模块），分别为“强度比较”模块和“空间精度比较”模块。两个模块的呈现顺序对每个被试进行随机化平衡（例如一半被试先做强度模块，另一半先做精度模块），以消除顺序效应。

- **Block设计与随机化**
  - **Block 1（强度/精度）**：包含20个Trials（10个刺激对组合 $\times$ 2次呈现顺序）。
  - **Block 2（精度/强度）**：重新随机化呈现这20个Trials。
  - **双盲控制**：实验者与被试在实验进行中均不知道当前呈现的是哪两个调制刺激条件。

- **单次试次呈现（Trial Procedure）**
    呈现“刺激A”（持续1.5秒） $\rightarrow$ 停止（持续1.0秒，用于感受野的机械和神经恢复） $\rightarrow$ 呈现“刺激B”（持续1.5秒）。不允许重复播放刺激，让被试通过第一直觉感受进行选择。

- **强制回答任务（2-AFC）**
    被试根据当前Block的任务要求，做出唯一的强制选择（不允许回答“一样”或“不确定”）：
  - **强度模块（Intensity Block）**：“刚才的两个刺激，哪一个的力度/压力更强？A还是B？”
  - **精度模块（Spatial Precision Block）**：“刚才的两个刺激，哪一个的触觉面积更小、更像是一个清晰的点？A还是B？”

#### 实验1参数设置

所有参数均由刺激频率200Hz、皮肤表面波传播波速以5m/s计算。

所有调制方法的轨迹中心点都为 (0, 0, 0.1) m

- DLM_2：选择的圆环半径为 0.625cm
- DLM_3：选择的圆环半径为 0.481cm
- ULM_L：选择的轨迹长度为 3cm (M=1.2)
- LM_L：选择的起终点距离为 1.5cm，每个周期内轨迹总路程为 3cm (M=1.2)
- LM_C：选择的圆环半径为 0.477cm (M=1.2)

##### 数据处理与理论验证

本研究采用基于贝叶斯推断的混合效应 Bradley-Terry 模型（Bayesian Mixed-Effects Bradley-Terry Model）对成对比较数据进行分析，以评估不同调制方式的主观强度和空间清晰度。

1. **数据预处理**
    - **数据整合**：读取所有被试的实验记录文件（CSV格式），剔除无效试次。
    - **格式转换**：将数据转换为**长格式（Long Format）**。每一行代表一次独立的成对比较（Trial），包含：被试ID（ParticipantID）、刺激A（StimulusA）、刺激B（StimulusB）以及二元选择结果（Outcome，1代表A优于B，0代表B优于A）。
    - **胜率矩阵（Win Matrix）**：生成 $5 \times 5$ 的胜率矩阵，直观展示各调制方式之间的胜负关系。

2. **统计模型构建**
    - **模型框架**：使用基于广义线性混合模型（GLMM）框架的 Bradley-Terry 模型。具体采用 **二项贝叶斯混合 GLM (Binomial Bayes Mixed GLM)** 进行参数估计。
    - **固定效应 (Fixed Effects)**：将5种调制方式（$DLM_2, DLM_3, ULM_L, LM_L, LM_C$）作为固定效应，用于估计每种方式的相对偏好得分（Log-odds Score）。
    - **随机效应 (Random Effects)**：将被试ID（ParticipantID）作为随机效应（Random Intercept），以解释不同被试在感知基准或判断标准上的个体差异（Subject Heterogeneity）。
    - **评分标准化**：模型输出的原始系数（Coefficients）经过中心化处理（Zero-centered），使得所有方法的平均得分为 0，便于直观比较相对优劣。

3. **统计检验与推断**
    - **成对比较 (Pairwise Comparisons)**：基于模型参数的协方差矩阵（Covariance Matrix），使用 **Wald 检验 (Wald Test)** 计算任意两种调制方式之间评分差异的 Z分数（Z-score）和 p值（p-value）。
    - **显著性水平**：
        - `***` : p < 0.001
        - `**` : p < 0.01
        - `*` : p < 0.05
        - `n.s.`: 无显著差异 (p ≥ 0.05)

4. **决策时间分析 (Decision Time Analysis)**
    - **目的**：评估不同刺激对之间的辨别难度（Discriminability）。较长的反应时通常意味着两个刺激的主观差异较小，难以区分。
    - **预处理**：
        - **无效数据剔除**：剔除过短（< 200ms）或过长（> 10s）的异常反应时数据。
        - **对数变换 (Log-transformation)**：对原始反应时进行自然对数变换，以校正反应时数据的偏态分布，使其更符合正态假设。
        - **Z-score 标准化 (Z-score Standardization)**：对每个被试的对数反应时（Log-RT）进行标准化处理（减去均值除以标准差），以消除不同被试基础反应速度的个体差异（Baseline Differences）。
    - **指标计算**：计算每对刺激组合的平均标准化对数反应时（Mean Z-Log-RT），构建反应时矩阵。

5. **结果可视化**
    - **相对评分条形图**：展示各调制方式在“主观强度”和“空间清晰度”上的相对表现（Log-odds 尺度）。
        - **误差棒**：展示 **95% 置信区间 (95% CI)**，计算公式为 $\pm 1.96 \times SE$。
        - **显著性标注**：使用连接线和星号标注具有统计学显著性差异的对比对。
    - **决策时间热力图**：使用热力图（Heatmap）展示各刺激对之间的平均标准化决策时间（Z-Log-RT）。颜色越深代表决策时间越长，区分难度越大。


#### 实验1结果

基于 Bradley-Terry (BT) 模型对成对比较数据进行分析，结果如下（评分表示 Log-odds 尺度下的相对偏好强度，±表示标准误差 SE）：

**1. 强度感知 (Intensity Preference)**

- **评分排序 (BT Score ± SE)**:
  - **ULM_L**: 0.72 ± 0.12 (95% CI: [0.48, 0.96]) —— **表现最佳**
  - **DLM_2**: 0.26 ± 0.12
  - **DLM_3**: 0.07 ± 0.12
  - **LM_C**: -0.14 ± 0.12
  - **LM_L**: -0.91 ± 0.13
- **显著性差异 (Wald Test)**:
  - **ULM_L**: 显著优于所有其他方法。
    - vs **DLM_2**: 差异显著 ($p=0.012$, \*)，优势比 (Odds Ratio, OR) 为 1.59（即 **ULM_L** 被选中的概率约是 **DLM_2** 的 1.6 倍）。
    - vs **DLM_3**, **LM_C**, **LM_L**: 差异极显著 ($p < 0.001$, \*\*\*)。
  - **DLM_2**:
    - 显著优于 **LM_C** ($p=0.031$, \*) 和 **LM_L** ($p < 0.001$, \*\*\*)。
    - 与 **DLM_3** 差异不显著 ($p=0.319$)。
  - **LM_L**: 显著弱于除 **LM_C** 以外的所有方法 ($p < 0.001$)，与 **LM_C** 相比也显著较弱 ($p < 0.001$)。
- **结论**: **ULM_L** 在呈现强度信息方面表现最佳，DLM_2 次之。ULM_L 能够提供比 **DLM_2** 更强的强度感知，且两者均显著优于传统的 LM 方法。

**2. 空间清晰度 (Spatial Clarity Preference)**

### 仿真1

#### 仿真1设计

本仿真旨在通过数值模拟验证实验1中涉及的五种触觉调制方式在软组织介质中的波动传播特性。仿真基于 **MATLAB R2025b** 平台及 **k-Wave** 声学仿真工具箱（`pstdElastic3D` 求解器），利用高性能工作站进行大规模并行计算。

**1. 仿真环境与物理模型**

- **计算域**：模拟尺寸为 $80 \times 80 \times 40$ mm³ 的三维空间，网格精度为 $1.0$ mm ($dx=dy=dz=1.0$ mm)。
- **介质参数**：模拟人体皮肤软组织。
  - 密度 ($\rho$): $923.5$ kg/m³。
  - 声速：剪切波速 ($c_s$) 设为 $5.0$ m/s；纵波波速 ($c_p$) 物理值为 $1540$ m/s，为提高计算效率，仿真中设定为 $100$ m/s (满足 $c_p \gg c_s$ 且不影响剪切波主导的触觉感知特性)。
  - 衰减：剪切波衰减系数 $\alpha_s = 10$ dB/(MHz$^y$ cm), $y=2.0$；纵波衰减系数 $\alpha_p = 0.1$ dB/(MHz$^y$ cm), $y=1.5$。
- **边界条件**：四周及底部设置 PML (Perfectly Matched Layer) 吸收边界，并配合 20 层 Sponge Layer 以消除边界反射，模拟半无限大空间。
- **时间步长**：根据 CFL 数 ($0.3$) 自动计算，确保数值稳定性。

**2. 激励源设置 (Excitation Source)**

- **源类型 (Source Type)**: 垂直应力源 ($S_{zz}$)，模拟聚焦超声产生的恒定声辐射力 (Constant Radiation Force)。
- **时间特性 (Temporal Profile)**: 源幅度在时间上保持恒定 (DC)，通过空间位置的变化产生调制效果。
- **扫描频率 (Scan Frequency)**: $f_{mod} = 200$ Hz (周期 $T=5$ ms)。
- **持续时间 (Duration)**: $75$ ms (15个扫描周期)。
- **空间分布 (Spatial Profile)**: 高斯分布点源 ($Gaussian$)。
  - **分布宽度**: 标准差 $\sigma = 4.25$ mm (代码参数 `ring_width`)。
- **强度 (Intensity)**: 峰值应力幅度设为 $100$ Pa。

**3. 调制方式与参数配置 (Modulation Methods & Configurations)**
本仿真对比了五种焦点扫描策略。针对每种模式，具体的几何参数与运动学参数设置如下（剪切波速 $c_s = 5.0$ m/s）：

1. **DLM_2 (Discrete Lateral Modulation - 2 Points)**: 焦点在圆周上2个离散点间切换，半径 $6.25$ mm。
2. **DLM_3 (Discrete Lateral Modulation - 3 Points)**: 焦点在圆周上3个离散点间切换，半径 $4.81$ mm。
3. **ULM_L (Uniform Lateral Modulation - Line)**: 焦点沿直线单向匀速扫描，长度 $30$ mm。(M=1.2)
4. **LM_L (Lateral Modulation - Line)**: 焦点沿直线双向往复扫描，长度 $15$ mm。(M=1.2)
5. **LM_C (Lateral Modulation - Circle)**: 焦点沿圆周连续匀速扫描，半径 $4.77$ mm。(M=1.2)

**4. 数据采集与分析**

- **观测平面**：记录介质表面下方 $1$ mm 深度的水平切面数据。
- **记录物理量**：应力张量分量 ($\tau_{xy}, \tau_{xz}, \tau_{yz}$) 及声压 ($p$)。
- **分析指标**：
  - **RMS Stress Map**: 稳态期间的应力均方根值分布，评估能量分布。
  - **Peak Stress Map**: 最大应力峰值分布。
  - **Rate of Change (Jerk) Map**: 应力随时间变化率的最大值，反映触觉刺激的动态变化强度。
  - **Spatial Gradient**: 应力场的空间梯度模，反映触觉的空间分辨率线索。
  - **Spatiotemporal Slice**: 提取中心截面的时空图 (XT-slice)，分析波动的传播与干涉模式。

**5. 可视化与数据后处理 (Visualization & Post-processing)**

为确保不同调制方式之间的公平对比及波场特性的准确呈现，可视化过程中采用了以下关键处理方法：

- **全局归一化 (Global Normalization)**: 对所有对比组（5种调制方式）的同类物理量（如RMS应力、峰值应力）采用统一的色标范围（基于所有组中的全局最大值），以直观反映相对强度差异。
- **频谱分析预处理**: 在绘制中心点波形的频谱图之前，对时域信号进行了 **去均值 (Demeaning)** 处理以消除直流分量，并施加 **汉宁窗 (Hann Window)** 以抑制频谱泄漏 (Spectral Leakage)，随后进行 FFT 变换并转换为单边频谱 (Single-Sided Spectrum) 的 dB 刻度显示。
- **时空图 (XT-Diagram) 增强**: 在绘制时空切片时，对每一空间点的时域信号进行了去均值处理，去除了静态背景应力，从而高亮显示动态波动的传播轨迹与干涉条纹。
- **波前快照 (Wavefront Snapshots)**: 提取首个周期内四个相位点 ($T/4, T/2, 3T/4, T$) 的瞬时波场，同样采用去均值处理以突出波动的正负极性（压缩/拉伸），使用发散色图 (Diverging Colormap) 清晰呈现波前的空间结构。
- **聚焦性能评估**: 提取焦平面上的峰值应力分布曲线，进行归一化处理，并计算 **半峰全宽 (FWHM)** 作为空间聚焦分辨率的量化指标。

**6. 计算平台 (Computing Platform)**

- **操作系统**: Ubuntu 24.04.3 LTS (Noble Numbat)
- **处理器 (CPU)**: AMD Ryzen 9 9900X (12-Core, 24-Thread, up to 5.6 GHz)
- **内存 (RAM)**: 64 GB
- **图形加速 (GPU)**: NVIDIA RTX5080 16GB GPU (Driver Version: 580.126.09)


#### 仿真1结果

以下分析均基于表面下 **1 mm** 深度切面的应力场结果；由于本仿真将纵波声速压缩为 **100 m/s** 以提高计算效率，因此**绝对应力量级更适合做相对比较，空间形态、频谱结构、传播模式和不同调制之间的相对差异更具有物理解释价值**。

##### `comparison_spatial_focusing 仿真结果`

- 从归一化空间聚焦曲线的 **FWHM** 可直接得到横向聚焦宽度排序：**LM_C = 24 mm < DLM_3 = 25 mm < LM_L = 28 mm < DLM_2 = 33 mm < ULM_L = 42 mm**。这说明在当前参数下，**连续圆周调制形成的剪切聚焦最紧，三点离散圆周已经几乎逼近连续圆周极限，而单向线扫最扩散**。简短讨论：这给出了“离散圆周逼近连续圆周”的直接证据。

- **LM_C** 与 **DLM_3** 的曲线几乎重合，且两者都有最窄的中心谷和最陡的双侧上升边缘，表明这两种模式在 `y=0` 截面上都形成了**以中心为参考、在两侧高度局域化的双叶型剪切场**。对你的项目背景而言，这与“圆心附近存在旋转剪切核心，主要应力能量沿轨道附近分布”这一理论是相符的。简短讨论：DLM_3 很可能已经进入与 **LM_C** 相同的“旋转剪切模态家族”。

- **DLM_3** 的 FWHM 仅比 **LM_C** 大 **1 mm**，这个差异已经接近当前仿真网格步长 **1 mm** 的量级，因此从图示结果上更合理的结论不是“两者显著不同”，而是**三点离散切换在空间聚焦上已近似实现连续圆轨迹的效果**。简短讨论：这对低刷新率、离散相位更新设备很有工程意义。

- **LM_L** 的 FWHM 为 **28 mm**，明显优于 **DLM_2** 的 **33 mm**，但又明显劣于 **LM_C/DLM_3**。这说明**双向线性扫描虽然能在轨迹两端附近形成较强局域峰值，但其整体空间约束能力仍不如旋转类轨迹**。从物理上看，这与 **LM_L** 中存在方向反转、尾迹重组和沿线段展开的波前有关。简短讨论：LM_L 更像“端点瞬态增强”，不是“最佳聚焦”。（同时需要考虑LM_L和LM_C的轨迹长度本身就不相同，这一条参考价值有限）

- **DLM_2** 的归一化曲线比 **DLM_3/LM_C/LM_L** 更宽，且在中心附近形成较宽的平台型低谷而不是尖锐窄谷，说明两点切换产生的主要响应仍集中在两个离散驻留点邻域，中间区域更多是**两点波场相遇后的弱干涉区**，而不是被压缩成紧致的旋转核心。简短讨论：DLM_2 的机制更偏“点间定时干涉”，不是旋转剪切场。

- **ULM_L** 的曲线最宽，半高宽达到 **42 mm**，比 **LM_C** 大约 **75%**。这说明即使其轨迹速度匹配剪切波速、可能有利于波前压缩，**其空间能量分布在稳态 RMS/归一化聚焦意义上仍然最分散**。简短讨论：ULM_L 的优势不在静态聚焦宽度，而应在动态传播组织。

- 五条曲线全部呈现**对称双峰 + 中心低谷**，而不是单峰聚焦于 `x=0`。这说明在当前切面和物理量定义下，真正被强化的不是几何中心点，而是**中心两侧的剪切应力带**；中心更像是符号翻转、相位过渡或旋转中心，而不是应力最大点。简短讨论：这支持用“表面波干涉带”而不是“静态焦点点压”理解超声触觉。

- 各方法的双峰位置都大致落在 **|x|≈6–15 mm** 范围内，但峰位外移程度不同：**ULM_L** 与 **DLM_2** 更靠外，**LM_C/DLM_3** 更靠近中心。这与几何轨迹尺度和调制连续性高度一致：轨迹越接近连续圆周、半径越小，主响应越向中心收缩。简短讨论：轨迹几何确实被清晰编码进了表面剪切场。

- 从曲线中心部分的陡峭程度看，**LM_C**、**LM_L** 和 **DLM_3** 的在中心处的边缘最陡，意味着它们在这个位置具有更大的横向空间梯度 `|\partial \tau / \partial x|`。对机械感受器而言，**梯度往往比均匀应力更能提供清晰边界和点状感**。简短讨论：这可能解释它们在空间清晰度上优于更扩散模式。

- **ULM_L** 虽然最宽，但其中心低谷反而最浅，归一化后中心仍有约 **0.4** 的残余水平，而 **LM_C/DLM_3** 的中心谷更深，仅约 **0.2–0.3**。这说明 **ULM_L** 在横截面上没有把能量“挖空”成清晰双叶，而是保留了较强的中央残余响应。简短讨论：这可能意味着 **ULM_L** 不利于形成最锐利的空间分界。

- **DLM_2** 的中心低谷比 **LM_L** 更宽、更平，显示其中心区是一个**较长的低梯度平台**；而 **LM_L** 的中心谷更尖锐，说明往复线扫虽然整体不如旋转类紧致，但在中心附近仍产生了更强的局部相消/分界结构。简短讨论：LM_L 的换向机制确实会重塑中心附近梯度。

- 从“离散逼近连续”的角度，这组图给出一个很有价值的模式：**从 DLM_2 → DLM_3 → LM_C，空间聚焦从宽双点型逐渐过渡到窄环形/旋转型**，不是简单线性增强，而是**模态性质改变**。简短讨论：这很适合作为论文中“离散轨迹拓扑改变波场模态”的关键结果。

- 这组归一化图与实验1主观“空间清晰度”结果只有部分一致：**LM_C/DLM_3** 在仿真中最窄，但实验中最优的是 **ULM_L** 与 **DLM_2**。因此应明确指出：**FWHM 仅刻画稳态横向能量收缩，不足以单独预测主观清晰度**；真实感知还受时间变化率、频率匹配、受体同步激活和边缘扫过时序影响。简短讨论：这正是本项目最有研究意义的“仿真-知觉解耦”现象之一。

##### `comparison_cross_section_rms 仿真结果`

- `y=0` 截面的 RMS 剪切应力曲线都呈现**双峰对称分布**，峰值不在 `x=0`，而在中心两侧形成两个主响应带；这说明稳态剪切能量主要沉积在**轨迹邻域或波前干涉带**，而不是几何中心。简短讨论：这再次支持“触觉来自表面剪切干涉带”的观点。

- 从绝对幅值看，峰值排序非常清楚：**LM_C** 最高，峰值约 **22–23 kPa**；**LM_L** 与 **DLM_3** 次之，约 **19–20 kPa**；**DLM_2** 约 **17 kPa**；**ULM_L** 最低，约 **10–10.5 kPa**。在当前仿真设定下，**连续圆周扫描在稳态 RMS 能量积累上最强，单向线扫最弱**。简短讨论：RMS 角度下 **LM_C** 是最强的稳态剪切能量源。

- **LM_C** 的两个峰最尖、最高、最靠近中心，峰位约在 **x ≈ ±6–8 mm**。这说明连续圆轨迹把剪切能量压缩在较小半径附近，形成了**最标准、最轴对称的高能旋转环带**。简短讨论：LM_C 可以作为旋转剪切稳态的基准模式。

- **DLM_3** 的峰值与 **LM_C** 很接近，但略低、略宽，峰位也接近 **±7–8 mm**。这表明三点离散切换已经能在稳态 RMS 上重建出**接近连续圆周的环状能量组织**，只是仍受离散切换引入的相位跳变和时间碎裂影响。简短讨论：DLM_3 是连续圆周的有效离散近似。

- **LM_L** 的峰值也接近 **20 kPa**，但两峰更宽、谷底更高，说明其高应力并非来源于稳定轴对称环，而更像是**沿线性往复轨迹两侧形成的宽双叶积累区**。这与其方向反转、端点换向导致的波场重排一致。简短讨论：LM_L 产生的是“强但不够紧”的稳态分布。

- **DLM_2** 的峰位明显更外侧，约在 **x ≈ ±11–13 mm**，且峰较宽。这直接反映了两点离散驻留的几何特征：主应力能量主要保留在两个目标点邻域，中间区域仅形成次级干涉。简短讨论：DLM_2 的能量拓扑是“双驻留点型”，不是“中心旋转型”。

- **ULM_L** 的 RMS 曲线最平缓，双峰位于更外侧，约 **x ≈ ±14–16 mm**，并且峰值只有其他方法的一半左右。这说明单向线扫在稳态平均意义下并不形成高局域应力累积，而更可能把能量分散为**沿传播方向展开的时空波前列车**。简短讨论：ULM_L 的关键物理量不应是 RMS 峰值。

- 中心 `x=0` 处的谷值排序同样有意义：**ULM_L** 中心谷最浅，谷值约 **4.3–4.5 kPa**；**DLM_2** 谷值约 **5 kPa**；**LM_L** 谷值约 **7 kPa**；**LM_C** 谷值约 **5.5–6 kPa**；**DLM_3** 中心谷值最小，最深，约 **4 kPa** 左右。更深的中心谷意味着中心附近更接近**相位过渡/相消区**，更浅则说明中间仍保留了较多背景剪切。简短讨论：中心谷深度是区分“紧致双叶”与“扩散扫过”的一个有用指标。

- 如果看**峰-谷对比度**，**LM_C** 最大，约 `(23-5.5)/23 ≈ 0.76`；**DLM_3** 也很高；**ULM_L** 最小。这意味着 **LM_C** 和 DLM_3 的横向边界最清楚，局部空间梯度最大；而 **ULM_L** 的空间轮廓最平。简短讨论：高峰谷对比度更有利于形成清晰的空间边界线索。

- **DLM_3** 与 **LM_C** 都表现出更接近“窄峰 + 深谷”的形态，而 **ULM_L** 更接近“宽峰 + 浅谷”。从机械受体驱动角度看，前者更像强局部边界刺激，后者更像大面积、低对比度的动态扫过刺激。简短讨论：它们可能对应两种完全不同的知觉编码策略。

- **LM_L** 的峰值虽然高，但中心谷也相对较高，说明其两侧高应力区与中心之间并没有形成像 **LM_C** 那样干净的分离。这提示 **LM_L** 在往返过程中可能把前缘、尾迹与换向瞬态叠加到中心邻域，使整体更“厚”。简短讨论：LM_L 的强度可能来自端点瞬态重组，而不是纯粹聚焦。

- **DLM_2** 的曲线最能体现“离散点”的物理特征：两个峰较远、峰间平台较宽、边缘衰减较缓。这种形态意味着其主效应不是连续轨迹上的压缩，而是**两个固定激励点之间的表面波定时叠加**。简短讨论：这与 **DLM_2** 的理论假设高度一致。

- 所有方法在 `|x| > 25–30 mm` 处都快速衰减到接近零，说明在该深度和频率下，表面附近可有效驱动的稳态剪切场是一个**尺度约数厘米的局域近场**，并未在横向大范围内保持高幅。简短讨论：这与皮肤表面波作为局域传播模态的预期相符。

- 这组 RMS 排序与实验1主观强度排序**明显不一致**：仿真中 **ULM_L** 最低，但实验中 **ULM_L** 最高；仿真中 **LM_C** 最高，但实验中并不占优。这是一个非常重要且可信的结果，因为它明确说明**触觉强度不等于稳态 RMS 剪切能量**。简短讨论：真正决定主观强度的更可能是动态同步、波前压缩和受体敏感频段匹配。

- 同样，这组图也提醒不能把“峰值高”简单等同为“更清晰”。例如 **LM_C** 的稳态聚焦与 RMS 都最好，但实验中的空间清晰度并未显著压倒 **ULM_L/DLM_2**。简短讨论：知觉清晰度需要引入时间维度来解释。

- 从研究价值上看，最值得强调的不是“哪条曲线最高”，而是这里已经暴露出两类调制机制：**LM_C/DLM_3/LM_L 属于高 RMS、较强局部积累型；ULM_L 属于低 RMS、但可能高传播组织度型**。简短讨论：这为后续把“稳态能量型”和“动态传播型”触觉机制分开建模提供了依据。

##### `RMS/Peak 仿真结果`

- 五种调制在稳态下都没有把最大剪切应力放在几何中心，而是在中心周围形成了**环状、轨道状或带状高应力区**，中心点普遍表现为局部低谷；这说明触觉主导刺激并不是“静态焦点压强最大点”，而是**表面剪切波在轨迹附近的时空干涉带**。这支持从表面波传播而不是静态声压峰值理解超声触觉形成机理。
  
- **DLM_2** 的 RMS 图和 Peak 图都表现为明显的**水平胶囊形双热点结构**，中心是一条狭长低谷，左右两端形成对称高值区；这说明两点切换主要在两个离散驻留位置附近积累能量，而中间区域更多承受的是**相位相消与波前交汇**。这与 **DLM_2** 的“点到点定时干涉”理论假设是相符的。

- **DLM_3** 的 RMS/Peak 图从 **DLM_2** 的“双热点”进一步收缩为**更紧致的近方圆形环带**，中心空洞更小，且存在轻微的方位不均匀与旋转偏置；这表明三点离散轨迹已经不再只是两个位置之间的往返叠加，而是开始形成**离散旋转剪切模态**。这说明 DLM 从 2 点到 3 点存在模态性质上的跃迁。

- **LM_C** 的 RMS/Peak 图最接近**轴对称环形分布**，中心低谷最圆整，外围高值带最均匀；这说明连续圆周运动能够在轨道附近持续产生方向连续变化的表面剪切波，从而形成最稳定的**旋转干涉环**。这使 **LM_C** 成为当前仿真中最标准的“旋转剪切场”参照组。

- **LM_L** 的 RMS/Peak 图表现为**沿 x 方向展开的双叶结构**，高值主要集中在中心上下两侧而不是整条线段

- 上述 RMS 幅值排序与实验1中的主观强度排序并不一致，尤其是 **ULM_L 在仿真中峰值最低，但实验中主观强度最高**；这表明**RMS/Peak 不是决定触觉强度的唯一主导量**，动态量如 **时间变化率、空间梯度、频率匹配、波前压缩** 很可能更关键。这个“仿真-心理物理不一致”是很强的研究切入点。

##### `Signed XT Diagram 仿真结果`

以下分析基于表面下 1 mm 深度、y=0 截面上的 signed shear stress \tau_{xz} 时空图（XT diagram）。横轴为时间（约 60–75 ms 的稳态窗口），纵轴为空间位置 x（mm），颜色表示 \tau_{xz} 的符号与大小：红色为正、蓝色为负、白色接近零。由于本图展示的是带符号的剪切应力分量而不是应力模长，因此其最重要的信息不是“总能量大小”，而是剪切方向、相位关系、波前传播轨迹、正负剪切对称性以及局域梯度结构。

- 五种调制在 \tau_{xz} 图上都呈现出“上下异号、中心近零”的基本结构：在 x>0 与 x<0 两侧形成符号相反的主剪切带，而 x≈0 附近普遍为窄的浅色过渡区。这说明该截面上主导的是一种关于中心近似反对称的横向剪切模式，即表面附近介质在中心两侧承受方向相反的切向-法向耦合剪切。简短讨论：这支持触觉线索更多来自剪切偶极/旋转型应变结构，而不是中心点的单纯压强峰值。

- 所有方法的主响应都集中在 |x|≈5–15 mm 的有限带内，而 |x|>20 mm 后迅速衰减接近零，说明在当前参数和深度下，tau_xz 的有效作用区域是一个贴近轨迹尺度的近场剪切带，并未在该截面上形成大范围扩展的强响应。简短讨论：这表明不同调制方式的差异主要体现在局部剪切组织方式，而不是简单的传播距离更远。

- **LM_C** 的图形最规整，表现为几乎理想的双带反对称结构：在整个稳态时间窗内，x>0 一侧持续为正、x<0 一侧持续为负，零交叉带稳定地锁定在 x≈0 附近，空间上近似平行、时间上缓慢演化。其峰值颜色接近色标上限，主带幅度约可达 ±2 kPa 量级。这说明连续圆轨迹在该截面上产生了最稳定的旋转剪切投影，相位结构高度一致。简短讨论：LM_C 可作为本研究中“稳定旋转剪切场”的标准参考模式。

- **LM_C** 在 x≈±5–8 mm 一带的主剪切带最强、最连续，且随时间从 60 ms 到 75 ms 有轻微增强趋势，说明圆轨迹连续扫描后，轨迹邻域建立了稳定的周期性储能/稳态叠加区。这与离散切换产生的局部跳变不同，更像连续输入下形成的准平稳表面波场。简短讨论：这类稳态结构可能更适合研究“持续旋转剪切”而不是瞬态冲击。

- **LM_L** 同样呈现明确的上下异号双带，但相较 LM_C，其主带更倾斜、更弯折、时间非平稳性更强：正负主带在约 68–71 ms 附近出现明显的形态转折，零交叉带也发生位置与曲率变化。这表明往复线性运动的方向反转会重构 tau_xz 的相位分布，使该截面上的剪切符号分布发生周期性重排。简短讨论：LM_L 的关键特征不是稳态强度，而是方向翻转诱发的相位重置。

- **LM_**L 在约 71–75 ms 时形成较强的右上正带（x>0）和右下负带（x<0），而在更早时刻（约 60–65 ms）其对应区域的幅值较弱，说明该方法存在明显的半周期不对称增强。这与双向往复扫描中“端点换向后，新前缘与旧尾迹相遇”的机制是吻合的：局部并不是匀速稳态，而是包含换向触发的强相位事件。简短讨论：LM_L 的研究价值在于换向瞬态，而不是平均场。

- **ULM_L** 的 tau_xz 图与其他四种方法最不同，表现为多条清晰的斜向传播条纹，这些条纹从左下向右上延伸，具有明显的有限斜率 dx/dt。在 XT 图中，这类斜纹直接对应于某种沿 x 方向传播的波前轨迹；斜率越大表示传播速度越快。图上主条纹横跨约 20–30 mm 空间范围、持续约 10–15 ms，量级对应的表观传播速度处于数 m/s，与仿真设定的剪切波速 5 m/s 同一量级。简短讨论：这是当前图组中最直接可见的“表面剪切波在传播”的证据之一。

- **ULM_L** 的斜向波前比其他方法更连续，说明单向匀速扫描在 y=0 剖面上更容易形成持续压缩并向前输运的波前列车，而不是局限于驻留点附近的局域剪切团块。其图形中的时间-空间倾斜关系，是“移动源速度与表面波速度匹配”这一理论假设最接近的可视化表现。简短讨论：**ULM_L** 最适合用来研究波前压缩和传播同步机制。

- **ULM_L** 的局部峰值颜色并不像 **LM_C** 那样饱和到接近全局最大，但它的传播条纹更长、更成体系，说明其优势并不在于局部 tau_xz 峰值极高，而在于形成了更明显的时空有序传播结构。这恰好提示：与触觉强度相关的关键量可能不是单点峰值，而是波前的到达同步性、梯度时序和受体群的顺序激活。简短讨论：这为解释“仿真峰值不最高但主观更强”提供了物理线索。

- **DLM_2** 表现出最典型的双驻留点主导结构：在 x≈+10 mm 与 x≈-12 mm 附近分别形成高幅主带，且两侧符号相反；而中心附近 x≈0 的响应较弱，仅有窄而断续的浅色/弱色区域。这说明两点离散切换主要在两个离散位置附近建立局部剪切场，中心并未形成连续稳定的旋转核心。简短讨论：**DLM_2** 更像“两个局域剪切源交替驱动”，而不是连续旋转场。

- **DLM_2** 的两个主带在时间上具有显著的块状驻留感，即色块在一定时间内近似停留于固定 x 位置，然后再发生切换后的重构。这是离散切换调制最直接的 XT 特征，区别于连续轨迹产生的斜带或平滑弯带。简短讨论：XT 图已经清楚证明 **DLM_2** 的动力学本质是“驻留—切换”，而不是“连续扫动”。

- **DLM_2** 在中心附近 x≈0 存在若干弱的正负交替小斑块，幅值远低于两端主带，说明从一个驻留点发出的波到达中心时，虽然确实会与另一驻留点相关波场发生相遇，但在当前 tau_xz 分量和该剖面上，这种中心相遇并未表现为占主导的高幅峰值，而更多体现为弱的交替剪切扰动。这与“中心处一定出现最大响应”的简单预期不同。简短讨论：**DLM_2** 的关键可能不是中心幅值最大，而是目标点附近的局部梯度增强。

- **DLM_3** 相比 **DLM_2**，主带从“两个固定驻留点”转变为更靠近中心、更加连续的三相循环重组结构：图中在 x≈±6–10 mm 附近存在较明显主带，同时中心附近 x≈0 也出现更丰富的弱-中等幅值符号切换。这说明三点离散切换已经不只是两个点之间的交替，而是在该截面投影上形成了某种离散旋转相位序列。简短讨论：**DLM_3** 相对 **DLM_2** 的本质变化是模态从“双点交替”向“离散旋转”过渡。

- **DLM_3** 的正负带整体上仍保持上下反号，但空间分布较 **DLM_2** 更紧凑，主带中心位置从约 ±10–12 mm 向 ±6–10 mm 收缩，符合其轨迹半径更小（4.81 mm vs 6.25 mm）的设定趋势。即使考虑波场扩展与卷积效应，XT 图仍能看出其主响应尺度确实随轨迹几何减小而内收。简短讨论：这说明几何轨迹尺寸确实被编码进了剪切场的主响应位置。

- **DLM_3** 在时间方向上出现比 **DLM_2** 更明显的细碎条纹与局部突变，反映一个周期内有更多离散相位事件。由于三点切换每周期包含更多焦点更新，图中出现更高时间复杂度是合理的。简短讨论：DLM_3 的潜在研究意义在于其可能引入更复杂的频率成分和受体时间编码。

- 从峰值相对幅度看，LM_C、LM_L 的主带饱和度最高，DLM_2 与 **DLM_3** 次之，ULM_L 单个局部峰值相对较低但条纹更长。这说明若只看该图中的 signed tau_xz 分量峰值，连续圆周和双向线扫更容易在该分量上形成大幅稳态或准稳态响应；但这并不等价于总剪切模长或主观强度的最终排序。简短讨论：必须区分“单分量峰值最大”和“触觉最强”这两件事。

- 五种方法的零交叉带都较窄，尤其 **LM_C** 和 **LM_L** 最清楚，表明在中心附近存在高的符号翻转梯度，也就是 \partial \tau_{xz}/\partial x 可能在过零位置很大。对皮肤机械感受而言，局部空间梯度往往比均匀应力更有刺激意义。简短讨论：过零线附近可能是比峰值点更值得关注的受体驱动区域。

- **LM_C** 的零交叉带最稳定且近似垂直于时间轴，说明其剪切分界面几乎固定在中心位置；相反 **LM_L** 和 **ULM_L** 的过零结构会随时间漂移或弯曲。这表明圆周连续运动更容易产生固定空间参考系下稳定的剪切分界面，而线性运动更容易产生移动分界面。简短讨论：稳定分界面与移动分界面可能对应不同类型的触觉知觉特征。

- **ULM_L** 图中的斜条纹在 x≈0 附近穿越时，并未形成特别强的局部驻留峰值，而是以较平滑的方式通过中心，这说明单向线扫并不依赖中心驻留增强，而依赖波前扫过时的时序组织。这与 DLM 的“在固定点附近积累”机制有本质不同。简短讨论：ULM_L 更像时序编码刺激，DLM 更像空间驻留编码刺激。

- **LM_L** 在约 68–70 ms 附近出现明显“X 形/折线形”过渡趋势，即原有主带与新主带的连接不是平滑平移而是带有换向痕迹的折转。这是双向往复轨迹的直接特征，说明速度方向的突然反转确实在时空图中留下了可辨认的动力学签名。简短讨论：这是 **LM_L** 与 **ULM_L** 的最关键区分证据之一。

- **DLM_2** 与 **DLM_3** 的图上可见若干近似竖直的小色块或短条纹，代表在固定 x 位置发生的快速符号/幅值更新；而 **ULM_L** 的主要结构则是斜纹，LM_C 的主要结构则是平行带。三者分别对应三类不同源动力学：离散驻留切换、连续平移传播、连续旋转稳态。简短讨论：仅凭 XT 图形态就可以对调制类型进行无监督分类。

- 在 tau_{xz} 这个分量上，LM_C 和 **DLM_3** 都体现出“旋转相关”的信号，但二者的差别很明确：LM_C 更接近连续相位旋转，带形平滑、几何对称性高；DLM_3 更接近离散相位跳转旋转，带形碎裂、局部不连续更多。简短讨论：这对研究“离散逼近连续旋转”的极限行为很有价值。

- **DLM_3** 虽然在宏观上接近 **LM_C** 的反对称双带结构，但其中心附近仍保留更多离散切换痕迹，说明“三点离散”尚未真正等效于“连续圆周”，只能说已进入同一种模态家族。简短讨论：DLM_3 是从离散模态过渡到连续旋转模态的临界候选。

- **DLM_2** 的主带位置明显比 **LM_C** 更远离中心，这不仅来自更大的轨迹半径，也提示两点切换更容易把主要剪切响应保留在两个目标点邻域，而不是向中心组织成稳定旋转场。简短讨论：DLM_2 更适合作为“点间干涉”研究对象，而不是“旋转剪切”研究对象。

- 如果把 XT 图理解为“在固定 y=0 剖面上看到的剪切相位投影”，则 **LM_C** 的时间平稳性最高、ULM_L 的传播性最强、LM_L 的换向特征最显著、DLM_2 的驻留性最强、DLM_3 的离散旋转特征最明显。这五种模式在图像层面上已经形成了彼此区分明确的动力学指纹。简短讨论：这为后续建立“调制方式—波场指纹—知觉结果”的映射提供了很好的表征基础。

- 从研究意义上看，当前最值得进一步量化的并不是单纯比较哪种方法更“红”或更“蓝”，而是从这些 XT 图中提取以下物理指标：波前斜率（传播速度）、零交叉线漂移、正负带面积比、局部符号翻转频率、换向瞬态持续时间、主带空间间距、过零带梯度宽度。这些量比单一峰值更接近不同调制策略的机制差异。简短讨论：下一步应把“图像观察”转成可统计的机制指标。

- 当前这组 tau_{xz} XT 图已经足以修正一个重要表述：不能把所有方法都简单理解为“中心处形成最大剪切”。至少在 y=0, z=1 mm 的 signed tau_{xz} 投影上，更普遍的事实是：中心是符号翻转带和相位分界区，而非主幅值峰区；主峰通常位于中心两侧。


### 结合实验1与仿真1的综合分析

- **DLM_3** 与 **LM_C** 对比，各项实验结果和仿真结果均非常接近。在心理物理学实验中两者在强度分数、精度分数上都非常接近，并且被试在区别两者时的反应时长比起其他刺激对显著更长，这个结果也表示两者非常相似，使用三个离散聚焦点“转移”即可实现绝大部分LM_C的效果，工程上可以用来指导刺激调制方式设计。LM_C的优势在聚焦点平滑移动，因此可以实现几乎没有可听频段的噪声。
- 但是DLM_3在两个分数上都略高于LM_C（但不显著），这可能表明由于DLM_3的离散控制，每个聚焦点停留时间都较长，从而可以充分给皮肤足够时间产生响应。而LM_C方法由于是连续聚焦点移动，可能使得皮肤上每个点来不及响应，到不了最大程度的响应，从而影响效果。
